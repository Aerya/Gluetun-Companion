Configuration Docker requise

Pour que le companion puisse contrôler Gluetun, vérifiez que votre docker-compose.yml du companion contient : 

volumes:
  - /var/run/docker.sock:/var/run/docker.sock
  - /chemin/vers/compose/gluetun:/compose

environment:
  GLUETUN_CONTAINER: gluetun-airvpn
  COMPOSE_DIR: /compose
  COMPOSE_PROJECT: nom_du_projet

Le companion écrira un docker-compose.override.yml dans ce dossier pour changer SERVER_NAMES et relancera le service avec docker compose up -d --force-recreate.
